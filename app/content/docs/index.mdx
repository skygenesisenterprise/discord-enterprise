Bienvenue dans la documentation Sky Genesis Enterprise.

Cette base est volontairement courte afin de fournir une structure exploitable :

- une route publique `/docs`
- une navigation latérale générée depuis `content/docs`
- un rendu MDX avec table des matières
- une recherche prête à évoluer

Ajoute tes pages dans `content/docs`, puis référence-les dans `meta.json` pour contrôler l'ordre de navigation.
