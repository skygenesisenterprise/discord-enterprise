## Organisation

Les pages de documentation vivent dans `content/docs`.

Chaque fichier MDX peut définir un titre et une description dans son frontmatter :

```mdx
---
title: Ma page
description: Résumé court de la page.
---
```

## Navigation

Le fichier `content/docs/meta.json` contrôle l'ordre des pages dans la sidebar.

## Prochaine étape

Crée des sous-dossiers par produit, API ou guide, puis ajoute un `meta.json` dans chaque dossier pour garder une navigation claire.
